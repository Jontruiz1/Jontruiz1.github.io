CatBot

A cat themed, web service bot programmed in Python for the popular communication platform called Discord. It was created using a wrapper API called Discord.py and is hosted on Render.com which provides automatic deploying from Github.

See the bot in action

The bot in action!

Hover over the images to play the gif

An integration with a MongoDB database to track the total number of times the =taco command has been entered. The database also tracks the individual user's taco count and announces it when the =mytacos command is entered. Also tracks the total times the =pet command is entered as well as the individuals count. When the individual count is above 10, the bot will send an image of a happy cat and otherwise will send a cat that's hissing/mad.

An integration with the OpenWeatherMap api that displays the weather of any city in the world using the =weather {city name} command. Also provides an image of a cat that fits the weather of that city.

Allows the user to play a game of Rock, Paper, Scissors by entering the =roshambo command. The user interacts with the emojis that are displayed and can either win, lose or draw with the computer's random choice. The player can either play 6 times in a row or choose to quit which will display the total number of wins, losses, and draws.
